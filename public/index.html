Cisecraft

Cisecraft is a Minecraft server hosted on the storm.cise.ufl.edu bastion server. In order to connect to the server, you must have SSH access to storm.cise.ufl.edu, which means you must be registered in the CISE department at the University of Florida.

How to play:

  1. (Recommended) Set up SSH public-key authentication if you have not already done so. Additionally, touch ~/.hushlogin on the server and put LogLevel error in your ~/.ssh/config (or pass -q to ssh) to shut up the annoying server banners.

  2. When you want to play, SSH into storm.cise.ufl.edu and forward TCP port 25452. The SSH session must remain in the background while you play. I highly recommend creating a script for this.

    ssh -q -L 25452:localhost:25452 -o ExitOnForwardFailure=yes username@storm.cise.ufl.edu

    For Windows, you can create a desktop shortcut. Right-click on the desktop, select New > Shortcut, then in the input box, type the full command line from above. Opening this shortcut will then automatically run this command in a terminal window.

    I have a script named connect-shell on the server with these contents:

    #!/bin/sh
    echo "You are now connected to Cisecraft."
    echo "Access the server at: localhost:25452"
    echo "Press CTRL-C to quit."
    exec cat >/dev/null

    So the command I use is:

    ssh -q -L 25452:localhost:25452 -o ExitOnForwardFailure=yes li.daniel@storm.cise.ufl.edu "exec cisecraft/connect-shell"

  3. In Minecraft, create a server with address localhost:25452. Start the SSH session, then connect to the server as usual.
