Cisecraft

How to join:

  1. Generate an SSH keypair:

    Linux: ssh-keygen -f ~/.ssh/id_cisecraft

    Windows: ssh-keygen -f %USERPROFILE%\.ssh\id_cisecraft

  2. Send me an email with the public key file (id_cisecraft.pub) to li.daniel@ufl.edu. Make sure you send the right file; never send the private key (id_cisecraft) to anyone!

  3. Once your key is configured, you can tunnel to the Cisecraft server using ssh. I highly recommend creating a script for this.

    Linux: ssh -q -L 25452:localhost:25452 -o ExitOnForwardFailure=yes -o BatchMode=yes -i ~/.ssh/id_cisecraft li.daniel@storm.cise.ufl.edu

    Windows: ssh -q -L 25452:localhost:25452 -o ExitOnForwardFailure=yes -o BatchMode=yes -i %USERPROFILE%\.ssh\id_cisecraft li.daniel@storm.cise.ufl.edu

    For Windows, you can download the script here: connect.cmd. Save to the desktop if you like. The first run might trigger an antivirus warning, just run anyway.

  4. In Minecraft, create a server with address localhost:25452. When you want to enter the server, first start the ssh session (make sure you keep it running in the background). If your key was configured correctly, there should be no password prompt, and you should receive a message saying the connection was successful. When you are finished playing, go back to the ssh session and press CTRL-C to terminate the connection.
